# OrangeBox

The Orange Box is an autonomous data-collection system specialized for an application in the [WatchPlant](http://watchplantproject.eu/) project.   
Orange Box is waterproof, battery-powered, and wireless. It is built around the Rock Pi S single-board computer running Ubuntu 22.04.

This repository forms the main structure of important system components:
* Configuration files
* Scripts for maintainance and automatic startup
* User interface
* Hardware interfaces and components managed by [gitman](https://gitman.readthedocs.io/en/latest/).
